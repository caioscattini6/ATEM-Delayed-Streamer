Machine B Setup Instructions

  1. Network Setup: Ensure Machine B can access Machine A's drive.
  2. OBS Scenes: You MUST create two specific scenes in OBS:
    • Create a scene named DELAYED_STREAM. Add your Media Source here, pointing to the shared .ts file. Uncheck "Local File", set Network Buffering to 2MB, and check "Hardware Decoding".
    • Create a scene named SAFE_SCENE. Add your holding graphic or "Please Wait" screen here.
  3. OBS WebSocket: In OBS, go to Tools > WebSocket Server Settings. Enable it, set the port to 4455, and set a password. Put these details into the App's configuration panel.
